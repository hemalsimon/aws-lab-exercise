---

## Ώθςε Step 1: Create Custom Policy

```json

  "Statement": [
    {
      "Sid": "S3Access",
      "Effect": "Allow",

        "s3:ListBucket",

      "Effect": "Allow",
      "Action": [
        "ec2:DescribeInstances",
        "ec2:RunInstances",

  ]
}
```

# Ώθπς LAB 2: Service έΗΤ Service (EC2 έΗΤ S3 using Role)

---

## Ώθςε Step 1: Create Custom Policy for S3

```json

  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
```

## Ώθςε Step 3: Create EC2 Instance

---

```bash

```

---

## Ώθςε Step 5: Validate Access
